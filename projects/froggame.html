Frog Game Prototype

2022

One of my childhood favorite games is the 1996 Frogger reboot. This prototype was an attempt to create a modern spiritual successor to that title, using many ideas from the later Frogger sequels as well as the most successful Frogger clone, Crossy Road. It contains fully working gameplay with support for various types of hopping, moving platforms, and slopes.

The source code for the untitled frog game prototype can be found on GitHub here: Frog Game Prototype Source Code Repository
