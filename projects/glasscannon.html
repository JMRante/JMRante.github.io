Glass Cannon

2021

My favorite game mode in classic arena shooters is instagib: pure simplicity, speed, and skill. I have always wanted to explore how this kind of gameplay would work in single player: hence the title of this project, Glass Cannon. Most of the work done on this unfinished project is for the level generator, which hooks together Metroid Prime-style rooms in a random configuration. The idea was to have instagib combat in a procedural Metroidvania structure.

The source code for the Glass Cannon prototype can be found on GitHub here: Glass Cannon Source Code Repository
